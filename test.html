Nuitka Binary Performance Test

This HTML file is used to test the Nuitka-compiled markitdown binary.

Test Purpose

The purpose of this test is to:

  • Verify that the Nuitka binary works correctly
  • Compare cold-start times with PyInstaller builds
  • Ensure all dependencies are properly bundled

Expected Output

When converted to Markdown, this file should contain:

  1. Properly formatted headings
  2. Bold and italic text formatting
  3. Lists (both ordered and unordered)

If you can read this as a blockquote in the Markdown output, the conversion was successful!

Code Example

def hello_world():
    print("Hello from Nuitka-compiled markitdown!")
